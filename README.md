## 🏭 Проєкт · 3D ЧАЕС з лічильником радіації

Сторінка `project.html` — інтерактивна 3D-сцена Чорнобильської АЕС у вигляді **до вибуху 1986 року**. Побудована з геометричних примітивів Three.js:

- 3-й і 4-й реакторні блоки (з внутрішнім реактором, що світиться)
- Головний корпус і машзала з трубопроводами
- Культова червоно-біла вентиляційна труба
- Дві градирні на задньому плані
- Огорожа території

### 🔦 Лічильник Гейгера

Реалістичний прилад, що реагує на позицію камери:
- **Формула радіації:** `I = 8000 / r²` µSv/h, де `r` — відстань від камери до реактора
- **4 зони небезпеки:** безпечна (<0.3) / помірна (<5) / підвищена (<100) / небезпечна (>100 µSv/h)
- **LED-індикація:** зелений / жовтий (блимає) / червоний (блимає швидко)
- **Звукові біпи:** через Web Audio API, частота пропорційна рівню радіації
- **Управління:** обертай мишею, зум — колесо миші, **клавіша M — вимкнути звук**

## 🧪 Запуск локально

```bash
cd natasha-page
python3 -m http.server 8000
# Відкрий http://localhost:8000
```

⚠️ **Важливо:** відкривай саме через `http://localhost`, а не `file://` — бо Three.js і Web Audio вимагають HTTP-контекст.
