ЧАЕС · 1986

3D-модель 4-го енергоблоку Чорнобильської АЕС у вигляді до вибуху. Обертай сцену мишею — лічильник Гейгера буде біпати інтенсивніше, коли наближаєшся до реактора. Чим ближче — тим вищий рівень радіації.

Про проєкт

Я люблю атмосферу серії ігор S.T.A.L.K.E.R. — українська історія, Зона, артефакти, тривожна музика. Вирішила зробити 3D-модель ЧАЕС у тому вигляді, якою вона була до аварії 1986 року, і додати інтерактивний лічильник Гейгера, який реагує на положення камери.

Стек: HTML · CSS · vanilla JavaScript · Three.js (r128) · Web Audio API. Без фреймворків, без бандлерів — усе вантажиться з CDN.

Технічні деталі

  • 01 3D-модель зібрана з примітивів Three.js: BoxGeometry для корпусів, CylinderGeometry для труб та вентиляції.
  • 02 OrbitControls обмежені — камера не може пройти крізь підлогу, мінімальна/максимальна відстань задані.
  • 03 Рівень радіації обчислюється як 1/r² (квадратичне спадання), де r — відстань від реактора до камери.
  • 04 Біпи генеруються через Web Audio API: OscillatorNode з квадратною хвилею, частота 850 Гц, модульована GainNode.
  • 05 Періодичність біпів пропорційна рівню радіації: чим ближче — тим частіше.
